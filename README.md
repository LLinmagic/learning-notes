## 实践流程
1. 安装Git并配置用户名、邮箱。
2. 创建本地仓库，提交三次代码并推送至GitHub。
3. 解决SSH密钥认证问题。

## 遇到的困难及解决
- **问题**：推送时报错 `Permission denied (publickey)`。
  - **解决**：重新生成SSH密钥并添加到GitHub账户。
